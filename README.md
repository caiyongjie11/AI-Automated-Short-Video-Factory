# AI Short Video Factory MVP

## 功能

- AI 文案生成
- AI 配音
- FFmpeg 自动生成视频
- React 前端

## 启动项目

### 后端

```bash
cd backend
npm install
npm run dev
```

### 前端

```bash
cd frontend
npm install
npm run dev
```

## 依赖

### FFmpeg

https://ffmpeg.org/download.html

### Edge-TTS

```bash
pip install edge-tts
```
